Rasberry Pi Case

The assignment was to construct a case for a Rasberry Pi. This was to help us learn to work with the program Fusion 360.


Here are the different steps I took in Fusion 360 in creating the box.


This is what it looked like when I printed it out.



The first time I attempted to print this, it did not work because I had it laying flat and the 3D printers are unable to create a hollow object.

The second time it worked because it had the hollow part facing up so that the 3D printer would just print around it. More the most part it worked,

but the Rasberry Pi did not fit inside because I made the dimensions way to small.
